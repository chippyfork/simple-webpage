Creating web pages is easy

I made this to show you how easy it is to create a webpage and publish your content online.

When creating a website, it's too easy to get bogged down with technical stuff. Which Content management system. Which website builder. It's easy to forget the basics.

This page is html, with a tiny bit of css to make it a bit easier on the eye. It shows how the basics of the web are still relevant today. And can help you get something live in a few minutes.

Here's why this page is awesome.

Lightweight

It's only 3kb! - As there's little resources, page load speeds are quicker. Nobody likes a slow loading page. And nobody likes loosing customers.

Accessible

As the page is simple it's easy to read and screen readers love it too. The key here is getting your message across. If you don't need that image, don't use it!

Responsive

This page works on any device. Html is responsive by default. Always has been. You don't need to create complicated grids and breakpoints to make the magic happen.
